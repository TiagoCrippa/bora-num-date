# Bora num date? 💜

Um convite interativo de "date" feito em uma página só — inspirado naquele trend do TikTok.
HTML/CSS/JS puro, sem dependências (só Google Fonts).

## ✨ O fluxo

1. **A pergunta** — 🥺 com o balão "PENSA BEM..." e os botões **SIM** / **NÃO** (o NÃO foge do mouse e do toque).
2. **Confirmação** — "VOCÊ DISSE SIM?".
3. **Data e horário** — escolha quando rola.
4. **Comida** — Hambúrguer, Sushi, Massas, Tacos ou Pizza.
5. **Vibe** — Golfe, Caminhada, Cinema, Dança, Parque ou Praia.
6. **Final** — 🌹 com o resumo do date montado a partir das escolhas.

## 🚀 Rodar localmente

É só abrir o `index.html` no navegador. Ou, se preferir um servidor:

```bash
python3 -m http.server 8000
# abre http://localhost:8000
```

## 🎨 Personalizar

- Cores: variáveis CSS em `:root` (topo do `<style>`).
- Opções de comida e vibe: arrays `foods` e `vibes` no `<script>`.
- Textos: direto no HTML de cada `<section class="screen">`.
